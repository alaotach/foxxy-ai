üì• Download ONNX Runtime Web

You need to download the ONNX Runtime Web library locally because Chrome extensions cannot load scripts from CDNs.

Option 1: Download Pre-built (Recommended)

  1. Visit: https://cdn.jsdelivr.net/npm/onnxruntime-web@1.17.0/dist/ort.min.js
  2. Right-click ‚Üí "Save As..."
  3. Save to: extension/libs/ort.min.js

Option 2: Using npm

cd extension
npm install onnxruntime-web@1.17.0
copy node_modules/onnxruntime-web/dist/ort.min.js libs/ort.min.js

Option 3: Using PowerShell

cd extension
New-Item -ItemType Directory -Force -Path libs
Invoke-WebRequest -Uri "https://cdn.jsdelivr.net/npm/onnxruntime-web@1.17.0/dist/ort.min.js" -OutFile "libs/ort.min.js"
